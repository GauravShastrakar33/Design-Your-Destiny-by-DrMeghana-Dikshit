5️⃣ EXACT SELECT IMPLEMENTATION (MANDATORY)
Use this logic in the Event form:

<Select
  value={form.requiredProgramCode}
  onValueChange={(code) => {
    const program = programs.find(p => p.code === code);
    if (!program) return;

    form.setValue("requiredProgramCode", program.code);
    form.setValue("requiredProgramLevel", program.level);
  }}
>
  <SelectTrigger>
    <SelectValue />
  </SelectTrigger>

  <SelectContent>
    {programs.map(program => (
      <SelectItem key={program.code} value={program.code}>
        {program.code} — {program.name}
      </SelectItem>
    ))}
  </SelectContent>
</Select>
✔ No empty values ✔ No sentinel hacks ✔ No Radix crash ✔ Admin-friendly ✔ Future-proof
